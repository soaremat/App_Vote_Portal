# App_Vote_Portal

Objetivo:
O principal objetivo de um app "Portal do Voto" direcionado a escolas é democratizar a educação política e estimular a participação cidadã entre os estudantes. Através de uma plataforma intuitiva e acessível, a ferramenta busca:

* Promover a cultura cívica: Incentivar a discussão sobre política e a importância do voto como ferramenta de transformação social.

 * Desenvolver o senso crítico: Estimular a análise crítica das informações e a formação de opinião sobre os candidatos e as propostas apresentadas.

 * Preparar os jovens para o exercício da cidadania: Equipar os estudantes com as ferramentas necessárias para que possam exercer seu direito de voto de forma consciente e responsável.
